# TGDAlarm
트게더 알리미 (GUI 없음)

<br/>
<br/>

# 다운로드 링크
https://github.com/SnowyWalk/TGDAlarm/releases

<br/>
<br/>

# 사용법
ids.txt에 알림을 받고 싶은 트게더 채널명을 적고 트게더 알리미 실행
<br/>
(ids.txt는 최초 실행 시 자동 생성됨)
